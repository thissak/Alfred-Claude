# 종목 종합 분석

종목코드를 받아 market.db + 네이버 증권 API에서 데이터를 수집하고 종합 분석 리포트를 제공한다.

## Step 1: 인자 파싱

`$1 $2` 형태로 인자를 받는다.
- 종목코드: 6자리 숫자 (필수)
- 종목명: 한글 이름 (선택)

종목코드 없이 종목명만 주어진 경우 `data/stock.json`의 watchlist에서 코드를 찾는다.

## Step 2: 분석 스크립트 실행

```bash
MARKET_DB_HOST=Ai-Mac-mini.local:8001 python3 scripts/stock_analysis.py <종목코드> <종목명>
```

스크립트가 수집하는 데이터:
- 기본정보 (시장/섹터/시총)
- 60일 일봉 추세 (MA5/20/60, 수익률, 고저점 대비)
- 밸류에이션 (PER/PBR/EPS/BPS/외인비율)
- 수급 20일 합계 (외인/기관/개인)
- 재무제표 최근 4기 (매출/영업이익/순이익/ROE)
- 공매도 최근 5일
- 네이버 뉴스 최근 10건
- 네이버 공시 최근 10건

JSON 결과: `data/stock-analysis/<종목코드>_analysis.json`

## Step 3: 종합 분석 리포트 작성

스크립트 출력과 JSON 데이터를 바탕으로 아래 구조로 분석한다:

### 3-1. 한줄 요약
현재 상태를 한 문장으로 요약 (예: "실적 개선 + 외인 매수 전환, 52주 저점 근처")

### 3-2. 가격/추세
- 현재가, 이평선 배열 (정배열/역배열/수렴)
- 고점 대비 하락률, 저점 대비 상승률
- 거래량 추세

### 3-3. 밸류에이션
- PER/PBR 수준 판단 (업종 대비 저평가/고평가)
- 외인비율 추세

### 3-4. 수급
- 외인/기관 매수세 방향
- 최근 5일 상세 수급 변화

### 3-5. 재무
- 매출/이익 성장 추세
- ROE, 부채비율 등 핵심 지표
- 분기 실적 흐름

### 3-6. 뉴스/공시 핵심
- 주가에 영향 줄 핵심 뉴스 3-5건 요약
- 중요 공시 사항
- 인포스탁/순위 기사 등 노이즈는 제외

### 3-7. 종합 판단
- 강점/약점 정리
- 매매 관점에서의 시사점 (진입/관망/주의)

## Step 4: 노션 저장 (선택)

사용자가 요청 시 Notion MCP로 결과를 저장한다.

1. 부모 페이지: "종목 분석 리포트" (없으면 생성)
2. 자식 페이지 생성: `{종목명} ({종목코드}) — {오늘 날짜}`
3. 분석 내용을 Notion 포맷으로 변환

## 규칙

- 뉴스 중 인포스탁 순위 기사, 광고성 기사는 자동 필터링
- 투자 추천/권유는 하지 않음 — 데이터 기반 분석만 제공
- surge 스킬과 차이: surge는 이상패턴(급등/급락) 탐지, 이 스킬은 종합 펀더멘털 분석
- 네이버 API는 비공식이므로 실패 시 KIS 뉴스로 폴백
