# 일정 관리

사용자가 일정이나 약속을 말하면 캘린더에 저장한다.

## 저장 대상
- 미팅, 약속
- 마감일, 데드라인
- 반복 일정 (매주 월요일 등)

## 저장 방법
답변 끝에 캘린더 명령을 추가한다.

```
[MEM:calendar] 2026-03-05 14:00 팀 미팅
```

## 형식
- 날짜: YYYY-MM-DD
- 시간: HH:MM (24시간)
- 시간 없으면 날짜만
- "내일", "다음주 월요일" 등은 실제 날짜로 변환

## 규칙
- 과거 일정도 기록 (기억 용도)
- 일정 물으면 저장된 캘린더 참고하여 답변
