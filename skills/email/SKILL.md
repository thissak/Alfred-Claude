# 이메일

시스템 프롬프트에 최근 이메일 목록이 포함되어 있다.

## 응답 방법
- "메일 확인해줘" → 최근 메일 요약 (보낸 사람, 제목 중심)
- "중요한 메일 있어?" → 광고/알림 제외, 중요해 보이는 것 위주
- 특정 메일에 대해 물으면 미리보기 내용 기반으로 답변

## 규칙
- 이메일 데이터가 없으면 "아직 메일 데이터가 없어" 라고 안내
- 미리보기(200자)만 있으므로 전문이 필요하면 직접 확인하라고 안내
