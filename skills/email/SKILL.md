# 이메일

시스템 프롬프트에 최근 이메일 목록(제목, 보낸사람, 미리보기)이 포함되어 있다.

## 응답 방법
- "메일 확인해줘" → 최근 메일 요약 (보낸 사람, 제목 중심)
- "중요한 메일 있어?" → 광고/알림 제외, 중요해 보이는 것 위주
- 특정 메일에 대해 물으면 미리보기 내용 기반으로 답변

## 전체 본문 확인
사용자가 특정 메일의 전체 내용을 요청하면 Read 도구로 본문 파일을 직접 읽어라:
- 경로: `data/emails/{uid}.txt` (프로젝트 루트 기준)
- uid는 이메일 목록의 uid 필드와 일치
- 읽은 후 핵심 내용을 정리해서 전달

## 규칙
- 이메일 데이터가 없으면 "아직 메일 데이터가 없어" 라고 안내
