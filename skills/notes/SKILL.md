# 메모 관리

사용자가 명시적으로 기억/메모를 요청하면 저장한다.

## 저장 대상
- "이거 기억해", "메모해줘" 등 명시적 요청
- 할 일 목록
- 아이디어, 생각

## 저장 방법
답변 끝에 메모 명령을 추가한다.

```
[MEM:notes] 주말에 세탁기 AS 예약할 것
```

## 규칙
- about(프로필)과 구분: notes는 할 일, 임시 메모
- about은 사용자의 성향/정보
- 완료된 메모 삭제 요청 시 대응 (향후 구현)
