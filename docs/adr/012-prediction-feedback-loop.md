# ADR 012: 주가 예측 피드백 루프 시스템

## Status
Accepted

## Context
급등 종목 분석 후 "다음에 오를 종목"을 체계적으로 예측하고 검증하는 루프가 필요.
단순 ML이 아닌, 해석 가능한 팩터 스코어링 + 자동 검증 + 가중치 조정 방식 채택.

## Decision
- 두 트랙 예측: Track A(테마 모멘텀) + Track B(눌림 가치)
- 6개 팩터 가중합 스코어링 (100점 만점, 65+ buy, 50+ watch)
- 5거래일 후 자동 검증 → 팩터별 적중률 기반 가중치 조정
- DB 테이블 3개: predictions, scoring_weights, factor_stats
- collector_daemon 파이프라인 말미에 검증→예측 순서로 실행
- 백테스트로 스코어링 v1→v4 진화 (v1 30% → v3 73% 적중률)
- 핵심 발견: "재료 강도"가 아닌 "재료 소화도"가 정방향 시그널

## Consequences
- 매일 장 마감 후 자동 예측 생성 + 과거 예측 검증
- 가중치가 시장 변화에 따라 자동 적응
- Track A는 시장 관심 종목, Track B는 소외 가치주 커버
- 감독님 전략(추격매수 + 눌림 진입)에 Track A가 더 적합
