# ADR 013: LightGBM 탑다운 스크리너 (미래주식)

## Status
Accepted

## Context
ADR 012의 GAN-style LLM 프롬프트 진화 시스템이 한계에 도달했다:
- 5종목 고정 테스트로 통계적 유의성 부재
- Discriminator 피드백이 자기모순 (v009 ETF 제외 → v010 ETF 재포함)
- 최고 성능 v009 P=60% R=50%도 과적합 의심
- 정배열 유지 ≠ 수익이라는 근본 문제

## Decision
LLM 프롬프트 진화를 LightGBM 기반 파라메트릭 모델로 대체하고, 3단계 탑다운 구조를 도입:

1. **Layer 1 시장 타이밍**: 전종목 정배열 비율로 BUY/CASH 판단
2. **Layer 2 섹터 필터**: 섹터별 정배열 비율+모멘텀으로 Hot/Cold 분류
3. **Layer 3 종목 선별**: LightGBM 듀얼 모델 (분류기+회귀기) → EV 기반 랭킹

종목 선별 모델:
- 분류기: P(20일 내 양수 수익) 예측
- 회귀기: E[20일 수익률] 예측 (Huber loss)
- 복합 스코어: EV = P(수익) × max(E[수익률], 0)
- 피처 25개 (MA건강도 8, 모멘텀 5, 거래량 2, 밸류 4, 재무 3, 시장 3)
- Walk-forward CV로 학습, 미래 데이터 누출 없음

## Consequences
- 전종목 30,809행 데이터셋으로 통계적 유의성 확보
- 백테스트 27개월: 월 +4.07%, 누적 +179.8%, 알파 +3.63%/월
- LLM은 종목 선별에서 퇴역, 서술(bull/bear case) 생성 역할로 전환
- investor_flow 테이블이 2개월치만 있어 수급 피처 사용 불가 (향후 축적 시 추가)
- 모델 재학습 주기 미정 — 월 1회 또는 레짐 변화 시 수동 재학습 예상
