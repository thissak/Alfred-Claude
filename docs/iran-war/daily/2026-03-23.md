### 원유 제재 면제의 함의
- 전쟁 상대국 석유를 풀어주는 전쟁은 **역사상 전례 없음**
- 트럼프 진짜 목표: obliterate가 아니라 **유가 관리 + 승리 서사**
- 이란도 면제를 거부·항의하지 않음 = **돈이 필요함** = 거래 가능
- 베센트: "이란 원유로 이란을 공격한다" — 명분 만들기

### 호르무즈 선별 통행의 함의
- 진짜 봉쇄면 모든 선박 차단해야 하나, 실제로는 "허가 기반 통행"

## 심층 분석: 인프라 MAD 구조와 미국 헤게모니

### 전력시설 타격 시 민간인 피해 추정

이라크 1991 전례 기반 (인구 5배 비례):
- 이라크: 발전소 20개 중 17개 파괴 → 5세 미만 47,000명 사망 (6개월)
- 이란 추정: 5세 미만 **10만~20만+**, 전체 초과 사망 **수십만**
- 이란은 이라크보다 취약: 5년 가뭄, 댐 저수량 14%, 제재로 의약품 부족
- 테헤란 1,500만 정전 → 72시간 내 물 위기 (펌프 정지)

### 가스전 vs 전력시설: 에스컬레이션 레벨 차이

| 구분 | 사우스파르스 가스전 (Level 3) | 전력시설 (Level 5) |
|------|---------------------------|-------------------|
| 성격 | 수출용 에너지, 이중용도 | **민간인 생존 인프라** |
| 국제법 | 회색지대 | 제네바 추가의정서 54조 보호 대상 |
| 행위 주체 | 이스라엘 (미국 거리두기 가능) | **트럼프 직접 선언** (책임 회피 불가) |
| 보복 수위 | 에너지 대 에너지 (카타르 LNG) | **생존 대 생존 (걸프 담수화)** |

레드라인은 Level 3과 5 사이에 존재한다.

### 인프라 MAD (상호확증파괴) 구조

```
냉전 핵 MAD:
핵무기 보유 → 양측 사용 불가 → 억지

중동 인프라 MAD:
미국 → 이란 전력 파괴 → 9,000만 물·전기 상실
이란 → 걸프 담수 파괴 → 1억 명 물 상실
    → 양측 사용 불가 → 억지 (= 블러핑의 구조적 근거)
```

사막 국가에서 전력·담수 시설은 생명유지장치. 핵이 없어도 핵급 억지력.

### 걸프 담수 위기 시 난민 규모

걸프 외국인 노동자 **3,300만 명** (전체 인구의 55%):
- 인도 ~900만, 파키스탄 ~400만, 방글라데시 ~300만, 필리핀 ~200만
- 영공·해상 모두 위험 → 물리적 대피 수단 없음 (NPR: "이미 갇혀 있다")
- 2008 미 외교전문: 리야드 담수 시설 파괴 시 **"1주일 내 대피"** 필요
- 쿠웨이트 식수 비축 수일, 카타르 수주 — 담수화 없이 생존 불가
- 시리아 난민 위기(660만)의 **10~50배 규모** 가능
