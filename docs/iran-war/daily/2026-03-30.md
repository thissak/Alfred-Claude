## 4. 삼각 교착 분석

```
이스라엘: 미사일 기지 29곳 파괴 + 핵시설 타격 완료 → "수 주 더 필요"
    ↕ (지상작전 요청 vs 독자 능력 한계)
트럼프: 지상작전 준비 ↔ 협상 동시 추진 = 이중 트랙의 딜레마
    ↕ (데드라인 4/6 → 또 연장?)
이란: 협상 거부 + 호르무즈·후티 레버리지 → "시간은 우리 편"
    + 후티 참전 → 이란 프록시 전선 4개로 확대
```

**Day 30 핵심 변화**: 공습→지상작전 단계 전환 논의. 전쟁의 질적 변화.
- 공습만으로는 호르무즈 봉쇄 해제 불가 → 카르그섬 등 물리적 점령 필요
- 지상작전 = 미군 인명피해 리스크 → 트럼프 정치적 부담 급증
- 이란 "침공하면 결전" 경고 → 에스컬레이션 사다리 한 단계 상승
