```
이스라엘: 핵시설 추가 타격 + "수 주 더" — 전쟁 종결 의지 없음
    ↓
트럼프: 데드라인 3회 연장 → 레버리지 소진. 루비오 "2~4주" = 출구 일정 공식화
    ↓                                            ↑
이란: 협상 거부 + 인도적 부분개방 = 강경 속 유연성 카드
    ↓

```

1. 이란 본토: 미국·이스라엘 공습 vs 이란 미사일·드론 반격

3. 걸프: 사우디·쿠웨이트·오만 민간 인프라 피격 본격화
4. 예멘(신규): 후티 3/28 첫 참전 + 바브알만다브 봉쇄 위협

핵심 구조 변화:
1. 후티 참전으로 예멘 전선 추가 — 이란 프록시 다전선 전통 패턴 완성
