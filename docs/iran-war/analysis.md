### 04-03 핵심 변화: 하라지 피격 + 트럼프 공허 연설 → 외교 채널 붕괴, 교착 심화
- **카말 하라지 자택 공습(4/1)**: 전 외무장관 중상, 부인 사망. 파키스탄 중재·밴스 VP 회담 준비 담당 — 유일한 외교 채널 사실상 무력화
- **트럼프 전국 TV연설(4/1)**: "전쟁 거의 완료, 2~3주 더" — 구체적 종전 계획 없음. 분석가 "한 달간 트윗 요약 수준"
- **"이란 대통령 휴전 요청" 주장**: 조건으로 호르무즈 개방 제시. 이란 즉각 부인
- **이란 미사일 파상공격**: 트럼프 "군사력 파괴" 선언 직후 텔아비브 타격 (14명 부상) — 모순 노출
- **유럽 이탈 가속**: 스페인·프랑스·이탈리아 미군작전 제한. UK 40개국 연합 호르무즈 개방 요구
- **민간 표적 확대**: 테헤란 의학연구소, 제약시설, 겜섬 담수시설, 반다르압바스 여객부두
- **KOSPI -4.47% 서킷브레이커**: 트럼프 연설 후 종전 기대 실망. 원/달러 1,519원
- 구조 변화: 외교 채널 파괴 → 4/6 데드라인 무의미화, 장기 소모전 vs 지상전 양갈래 수렴

### 03-31 핵심 변화: 지상전 시그널 + 파키스탄 중재 구체화 → 4/6 분기점
- **미군 7,700명 증파**: USS Tripoli 해병 3,500 + 82공수 2,000 + 해병 2,200. Pentagon "수 주 지상작전 준비"
- **트럼프 이중 메시지**: "obliterate" 위협 + "pretty sure of a deal" = 최대 위협과 최대 낙관 동시
- **이란 협상 부인**: "미국과 직접 협상 진행한 적 없다. 트럼프는 기만적" — 파키스탄 중재와 모순
- **파키스탄 4개국 회동**: 사우디·이집트·터키 외교장관 (3/29) → "수일 내 직접대화 호스팅" 선언
- **테헤란 정권 인프라 집중 타격**: IRGC 군수산업, 미사일 부품, UAV 엔진 시설. 고아원 피격 민간인 2명 사망
- **걸프 민간 피해 확대**: 쿠웨이트 발전소 인도인 노동자 사망, 바레인·UAE 알루미늄 공장 피격
- **후티 연속 참전**: 드론 2기 추가 발사 (요격). 일회성 아닌 지속 참전 확인
- **브렌트 월간 +55%**: 1988년 이후 역대 최대 월간 상승률. $111~115 레인지
- 구조 변화: 공습 한계 → 지상전 준비 전환, 파키스탄 중재 유일한 외교 채널

### 03-28 핵심 변화: 후티 참전 + 데드라인 재연장 → 다전선 소모전 전환
- **후티 첫 참전**: 이스라엘 남부에 탄도미사일 일제사격. 바브알만다브 봉쇄 검토 시사
- **트럼프 데드라인 4/6 재연장**: 48h→5일→10일, 블러핑 3회 확정. "협상 매우 잘 진행 중"
- **15개 조항 평화안**: 위트코프 → 파키스탄 → 이란 전달. 이란 "최대주의적" 거부 + 5개 역제안
- **루비오 G7**: "전쟁 2~4주 더. 지상군 불필요." G7에 전후 호르무즈 태스크포스 요청→동의
- **걸프 민간 인프라 피격**: 쿠웨이트 공항(레이더 파손·폐쇄), 오만 살랄라항(드론), 프린스술탄 기지(미군 12명 부상)
- **핵시설 추가 타격 (3/27)**: 아라크 중수로 + 야즈드 옐로케이크 공장 (이란 유일)
- **이란 인도적 통과 허용**: 호르무즈 통해 인도적 지원·농산물 "촉진·신속화" — 봉쇄 후 첫 부분 개방
- 구조 변화: 삼각→사각 교착, Tacit Freeze 후퇴, 장기 소모전 확률 상승

### 03-23 핵심 변화: 5일 유예 — Tacit Freeze 첫 공식 단계
- 트럼프, 이란 에너지/발전소 타격 5일간 유예 + "매우 생산적인 대화" 진행 중 발표
- 48시간 최후통첩 → 5일 유예 = 트럼프가 먼저 눈 감았다 (블러핑 확인)
- 시장 즉각 반응: 브렌트 $113→$100.50 (-12%), 다우선물 +815pt
- 네타냐후: "비밀 협상 하냐" 백악관 직접 확인 — 가장 불안한 당사자

### 03-22 핵심 변화: 미국 출구 전략 가시화
- 트럼프 톤 변화: 작전 축소 고려, 이스라엘 거리두기, 호르무즈 책임 전가

- 이란산 원유 1.4억 배럴 판매 허용 = 유가 안정화 + 이란 당근
- 핵심 난제: 협상 상대 부재 (지도부 다수 암살), 배상 요구 vs 동결자산 반환 우회로

## 인프라 MAD 구조 (03-23 신규)

핵무기 없이도 핵급 억지력이 작동하는 구조. 사막 국가에서 전력·담수 시설은 생명유지장치.

```
미국 → 이란 전력 파괴 → 9,000만 물·전기 상실 → 수십만 사망
이란 → 걸프 담수 파괴 → 1억 명 물 상실 → 국가 기능 붕괴
    → 양측 모두 실행 불가 → 블러핑의 구조적 근거
```

### 에스컬레이션 사다리 (레드라인 위치)
- Level 1~2: 군사시설·핵시설 타격 (합법~논란)
- Level 3: 에너지 수출 인프라 — 사우스파르스 (이중용도 회색지대) ← 현재
- **━━ 레드라인 ━━**
- Level 5: 전력시설 — 제네바 추가의정서 54조 "생존 불가결 시설" 보호
- Level 6~7: 댐·핵발전소 — 명시적 금지

### 시간 비대칭
- 트럼프 데드라인: 4월 중순 (미국 비료 파종), 11월 (중간선거)
- 이란 데드라인: 없음 — 호르무즈를 쥐고 버티면 시간이 갈수록 유리
- 이란은 양보할 구조적 필요 없음 — "가만히 있으면 이긴다"

### 미국 헤게모니 5대 기둥 손상 (진행 중)
1. 군사 신화: 중급 국가가 4주째 항전 + 비대칭 억지 실전 시연
2. 동맹: NATO·인태 불참, 걸프 균열, 트럼프 "겁쟁이" 발언으로 이탈 가속
3. 페트로달러: 이란 "위안화 결제 시 통과", 걸프국 안보 합의 근본 흔들림
4. 도덕적 리더십: 글로벌 사우스 "제국주의" 프레임, 전력 타격 시 "전쟁범죄"
5. 안보 보장: GCC 6개국 수도 전부 피격 — "미국이 지켜준다" 신뢰 붕괴
